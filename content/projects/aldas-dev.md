My personal corner of the internet. Built with Nuxt 4, terminal aesthetics, and a sprinkle of personality.

Features an interactive CLI mode where you can navigate the site using actual terminal commands like `ls`, `cat`, and `cd`.

The site pulls in my latest tweets from the X API and displays projects and interests from markdown files using Nuxt Content.
