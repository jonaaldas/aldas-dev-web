Open source is how I learn best — reading other people's code, contributing fixes, and building tools that anyone can use.

## Philosophy

Ship early, iterate often, document everything. The best open source projects are the ones that respect the user's time.

## Contributions

I try to contribute to the tools I use daily. Even small fixes — typos in docs, edge case bugs — make the ecosystem better for everyone.
